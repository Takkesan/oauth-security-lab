# Seccamp 2026 mini OAuth 2.0 & OpenID Connect

Test app before hands-on

## How to run locally

### Prerequisite

* node.js 24
* docker
* docker-compose
* gcloud
* java 26+
* firebase cli 15.15.0+

### Install

```shell
npm i
```

### Bulid

```shell
npm run build
```

### Run

```shell
npm run emulator & npm run dev
```

``` shell
cd docker
docker-compose up
```

Open [http://localhost:8080](http://localhost:8080).
